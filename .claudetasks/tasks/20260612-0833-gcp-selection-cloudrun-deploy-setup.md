**1. クラウド選定検討書（DOC-IF-001）作成:**
- Web調査（Cloud Run vs Fargate 課金モデル、GCP IoT Core 廃止、BigQuery/Looker Studio 料金）を踏まえ GCP vs AWS を比較
- 結論: **リテール事業系は GCP 採用を推奨**
  - コスト: Cloud Run ゼロスケールで月額約半分（GCP 4-5千円 vs AWS 7-8千円、ALB固定費が差）
  - 認証: Google Workspace × IAP で社内SSOが最小工数
  - 分析: BigQuery + Looker Studio がリテールBIに直結（ほぼ無料で開始）
  - AI: 画像認識パイプラインのクラウド化先として Vertex AI が自然
- トレードオフ: GCP IoT Core 廃止済（フリート管理が必要なら AWS IoT Core 部分採用を再検討）、メールは SendGrid 等で代替、fieldup との2クラウド併用は事業ライン単位で境界を明文化
