## 📝 作業サマリー

製品モデル詳細ページの BOM構成表で、既存の部品行が編集できなかった問題を解消し、複製ボタンを追加した。

**BomLineFormDrawer を追加/編集/複製の3モード対応に拡張:**
- 編集: 既存行の **部品マスタ（部品名）・数量・区分（オプション部品）** をすべて変更可能（useUpdate PATCH。バックエンドは ModelViewSet で対応済みのためフロントのみの変更）
- 複製: 元の行の値（部品・数量・区分）を初期値に新規作成するモード。「複製して追加」ボタン。同一部品のまま保存すると unique_together（1モデル1部品1行）で重複エラーになるため、ドロワー内に注意書きを表示

**ProductModelDetailPage:**
- BOM行を**行クリックで編集**、操作列に「編集 / 複製 / 削除」ボタン（各ボタンは stopPropagation で行クリックと干渉しない）
- ドロワー状態を判別可能ユニオン（create / edit / duplicate）で管理
- 基本情報セクションに **ファミリ / グレード / バリエーション** 表示を追加（4層分類導入後に詳細ページへ未反映だったもの）
